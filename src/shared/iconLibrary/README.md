Welcome to Eclair DS - Icons.

## Installation

Install using `npm`:

```sh
npm install eclair-icons --save
```

Install using `yarn`:

```sh
yarn add eclair-icons
```

## Usage

You can import icons in your Typescript or Javascript React project.

Using esm modules import & export:

```js
import { IconPaypal } from 'eclair-icons';
```

Using commonjs require & module.exports:

```js
const IconPaypal = require('eclair-icons/lib/commonjs');
```
